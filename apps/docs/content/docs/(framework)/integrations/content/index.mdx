## Introduction

**xyzdocs is very flexible.** You can integrate with any content source, even without an official adapter.

### Examples

You can see examples to use xyzdocs with a CMS, which allows a nice experience on publishing content, and real-time update without re-building the app.

- [BaseHub](https://github.com/fuma-nama/xyzdoc-basehub)
- [Sanity](https://github.com/fuma-nama/xyzdoc-sanity)
- [Community: Payload CMS by `MFarabi619`](https://github.com/MFarabi619/xyzdoc-payloadcms)
- [Community: Payload CMS by `bapspatil`](https://github.com/bapspatil/xyzdoc-payload-template)

Not enough? [Build your own content source](/docs/integrations/content/custom).
