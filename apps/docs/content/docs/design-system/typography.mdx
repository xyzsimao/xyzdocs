## 字体

字体配置包含字体、字号、字重和行高多个部分，下面分别进行介绍。

### 字体风格

字体用于设置文本呈现的风格，预设了三种字体风格，包括无衬线字体、衬线字体和等宽字体。下面为三种字体的直观展示：
    <Example> 
       <div className="items-center gap-2 row">
          {fontFamilies.map((font) => ( 
            <div key={font.code}>
          <div className="items-center gap-2 row"> 
          <strong className="font-mono uppercase text-brand ">
      {font.code}</strong>
                <span className="opacity-50">—</span>
      <strong>{font.name}</strong> <span className="ml-4 text-sm opacity-50">{font.desc}</span>
          </div>
     <pre key={font.code}  className={cn('text-lg mt-2 font-normal', font.code)}>The quick brown fox jumps over the lazy dog.
     <br/>
白日依山尽，黄河入海流。欲穷千里目，更上一层楼。</pre>  <br/> </div>  ))}
    </div>
     </Example>
